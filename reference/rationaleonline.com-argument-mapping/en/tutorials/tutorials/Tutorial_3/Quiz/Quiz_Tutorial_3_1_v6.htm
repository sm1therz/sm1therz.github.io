Tutorial 3 Quiz

Multiple-choice exercise

Choose the correct answer for each question.

 

  1. A multi-reason argument...

    1.   has at least one reason and at least one objection bearing directly upon the same contention
    2.   has at least two reasons bearing directly upon the same contention
    3.   has at least two reasons or objections bearing directly upon the same contention
    4.   has at least two contention

  2. Which of these is FALSE?

    1.   Every counter-argument is a multi-reason argument.
    2.   Every dispute is a multi-reason argument.
    3.   Every dispute involves counter-arguments.
    4.   Every counter-argument is part of a dispute.

  3. A dispute is...

    1.   a disagreement between people
    2.   any complex argument involving both reasons and objections
    3.   an argument in which there are both reasons and objections bearing upon a single contention
    4.   an objection to an argument.

  4. "Strange bedfellows" are

    1.   two reasons pointing at the same contention when they should be pointing at different contentions
    2.   two premises diagrammed as belonging to one reason when in fact they belong to different reasons
    3.   two premises in one reason which don't hold hands with each other
    4.   reasons which point to different contentions

  5. To strand co-premises, you should:

    1.   stick them in different reasons
    2.   stick one in a reason and one in an objection
    3.   stick them in the same reason even though they don't belong together
    4.   leave them unconnected to anything

  6. This argument map schema  depicts...

    1.   a multi-reason argument
    2.   a dispute
    3.   counterarguments
    4.   a multi-reason argument, a dispute, and counter-arguments
    5.   a dispute and counter-arguments, but not a multi-reason argument

  7. This argument map schema

    depicts...

    1.   a multi-reason argument
    2.   a dispute
    3.   counter-arguments
    4.   strange bedfellows
    5.   stranded co-premises

  8. What is wrong with this argument map?

    1.   Stranded co-premises
    2.   Strange bedfellows
    3.   Nothing
    4.   Yoda doesn't love bacon
    5.   It is not a dispute

  9. What is wrong with this argument map?

    1.   Stranded co-premises
    2.   Strange bedfellows
    3.   Iraq DOES have weapons of mass destruction.
    4.   It is missing a co-premise.
    5.   Nothing
