Exercise 5.5 Model Answer
 
Task

Produce an argument map of all the reasoning on p.3 of Apollo Moon Landings. 

If any of the objections are inference objections, make sure that your map shows them as objections to co-premises.

 
Model Answer

 
Discussion

There are two rebuttals, and they are both inference rebuttals, targeted on the same co-premise.

There is a missing layer between the primary objection and the main contention, which is not diagrammed here. 

The reason at the bottom right is a borderline case of an evidential reason.  It might also be regarded as not evidence for its contention, but rather an explanation of it.
