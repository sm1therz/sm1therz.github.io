Exercise 4.1 Model Answer
 
Task

Create an argument map of the following reasoning:

Main contention: The film used in the Apollo cameras would have melted.

Primary Reason: The mean surface temperature on the Moon is 107 degrees in the daytime Sun.

Rejoinder: With no atmosphere there is no convective heat transfer, only radiant and conductive. 

[based on 8.1]

Note: for this exercise you need not diagram hidden premises.
