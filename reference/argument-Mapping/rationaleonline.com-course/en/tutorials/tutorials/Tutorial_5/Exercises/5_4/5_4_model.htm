Exercise 5.4 Model Answer
 
Task

Consider this passage:

Hoax believers object to the claim that Apollo astronauts landed on the Moon on the grounds that NASA can't produce pictures from the Hubble Space Telescope of the equipment the astronauts left behind.  However, it is unlikely that any photographic evidence that NASA produced would convince die-hard conspiracy theorists. [based on 9.4]

  1. Decide what kind of objection the second objection in this passage is.
  2. If it is an inference objection, produce an argument map showing the inference objection as an objection to a co-premise.
 
Model Answer

The objection is an inference rebuttal.

 
Discussion

This inference rebuttal is similar to the one in the previous exercise, but it attacks a different co-premise of the primary objection.
