Exercise 3.3 Model Answer
 
Task

Produce an argument map of the reasoning in this passage:

Astronauts travelling through the Van Allen Belt would have had an exposure of between 1 and 2 rem. Radiation sickness symptoms do not start to show until around 25 rem. Thus it is reasonable to assert that they would not experience any immediate health problems that would prevent the journey to the Moon. [Based on 6.6]

Note that for this exercise your argument map need not represent any hidden co-premises.

 
Model Answer

 
Discussion

This is not a multi-reason argument; it is a simple argument in which two of the co-premises were explicitly stated.
