Quiz - Tutorial 2

The quiz is designed to help you learn the material.  In all cases, you should choose the best answer.  In many questions, there are answers which are partially correct, but are not the best response.

Quiz scoring is based on the number of answers you select, not simply on the number you get correct first time. You can get partial credit for selecting the correct answer with subsequent guesses.

Brower note. If you encounter any problems with quiz formatting or functionality, you may need to upgrade your web browser.  Recommended browsers are
Windows Windows Internet Explorer 6+
Macintosh Netscape 7+.  Note that Mac Internet Explorer v.5 does not work.
